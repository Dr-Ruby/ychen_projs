Raspberry Pi Car with QR Code Detection

A collaborative project by Yizuo Chen and Joshua Lee. This car is powered by a Raspberry Pi, uses a Pi Camera with OpenCV and C++, and identifies QR codes to launch BB balls. The system operates wirelessly using servers.

AC to DC Power Supply

This project involves designing and building an AC to DC power supply. The schematic was created in KiCad, the circuit board was printed, and all components were soldered and assembled. The power supply provides a voltage range from -20V to 20V, adjustable using knobs.

Color Sorting Ball Machine

A project using a Raspberry Pi, C++, and OpenCV to sort balls of different colors into separate bins. This system was designed and implemented to demonstrate object detection and real-time sorting capabilities.
