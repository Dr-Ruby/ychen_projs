Projects

Raspberry Pi Car with QR Code Detection

A collaborative project by Yizuo Chen and Joshua Lee. This car is powered by a Raspberry Pi, uses a Pi Camera with OpenCV and C++, and identifies QR codes to launch BB balls. The system operates wirelessly using servers.

AC to DC Power Supply

This project involves designing and building an AC to DC power supply. The schematic was created in KiCad, the circuit board was printed, and all components were soldered and assembled. The power supply provides a voltage range from -20V to 20V, adjustable using knobs.

Color Sorting Ball Machine

A project using a Raspberry Pi, C++, and OpenCV to sort balls of different colors into separate bins. This system was designed and implemented to demonstrate object detection and real-time sorting capabilities.

Ultrasonic Radar Module (Collaborative Project with Ryan McKay)

This collaborative project implements a real-time radar visualization system using a DE0-Nano-SoC FPGA board, an ultrasonic sensor (HC-SR04), and a servo motor. The radar data is transmitted via UART to a Python visualization interface that displays detected objects in a semicircular sweep, similar to a heartbeat sensor. Developed using SystemVerilog and Python (OpenCV).

Ongoing Capstone Research & Projects

Social Robot for Pediatric Anxiety Relief

A research collaboration with BC Children’s Hospital Research Institute and the UBC Faculty of Medicine, developing an interactive NAO robot to support pediatric patients during medical procedures. The system integrates speech recognition (Vosk), multimodal interaction, and emotional engagement analysis.

Student ID Attendance System (Real-Time Embedded Systems)

A collaborative project by Yizuo Chen and Ryan McKay. this project builds a student attendance tracking system based on NFC (Near Field Communication) using Raspberry Pi Pico 2W. It records student IDs and transmits data wirelessly to a remote server for real-time logging.

Face Detection System (Digital Image and Video Processing)

An image processing project focused on real-time face detection using OpenCV and MATLAB. The system is trained to identify faces in video streams and explore techniques such as Haar cascades and deep learning-based detectors.
